Rental Options

We offer several packages to suit any of your rental needs. Our packages are a tent, round tables with chairs, and banquet tables. Packages are not required and any of these services can be rented individually.

20X20 Tent Package

This package proivides a 20' by 20' frame tent with four round tables, fourty chairs, and two banquet tables. The 20X30 package services up to forty people.

20X30 Tent Package

This package proivides a 20' by 30' frame tent with six round tables, sixty chairs, and two banquet tables. The 20X30 package services up to sixty people.

20X40 Tent Package

This package proivides a 20' by 40' rope and pull tent with eight round tables, eighty chairs, and two banquet tables. The 20X40 package services up to eighty people.
